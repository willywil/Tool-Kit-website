Getting Started with Arduino

Author: Jon Wilhelm

Are you interested in building robots? I am, building robots is my hobby, my passion, and I hope one day my career. A robot consists of three things, the body (mechanical), electrical aka motors, and computers aka the brain. In this instruction I will guide you through programming of the brain. A computer is a device that processes 1s and 0s to perform a logical operation. In order to understand how to program a computer, I choose the Arduino to demonstrate this process. For our first program we will blink a LED (Light Emitting Diode).

Required Supplies:

  • Arduino UNO board
  • Arduino Software
  • USB cable that will fit the Arduino
Step 1: For this step, first download the Arduino software from Arduino.cc and then install the software after download is complete.

Step 2:

Upon completion of install, connect the USB cable to the Arduino and connect the opposite end to an available USB port on your computer. At this time your computer may recognize the Arduino and install available drivers. If your computer does not notify you it has recognized the Arduino with sound and/or a prompt, check the software. To verify your Arduino is connected in the software, first open the software, select “Tools” in the menu bar, then “Serial Port” you should have at least one item in the dropdown list. If the software did not recognize the board and does not display a serial port under “Tools” please seek help in Arduino.cc Forum.

Step 3:

The blinking of a LED is the common method of testing electronics’ functionality. In the empty space copy the following text. Use COPY and PASTE if needed. // // the setup function runs once when you press reset or power the board void setup() { // initialize digital pin 13 as an output. pinMode(13, OUTPUT); } // the loop function runs over and over again forever void loop() { digitalWrite(13, HIGH);   // turn the LED on (HIGH is the voltage level) delay(500);              // wait for a second digitalWrite(13, LOW);    // turn the LED off by making the voltage LOW delay(500);              // wait for a second } //End of Program //

Everything on the line after “//” represents comments and are not included in the program.

Step 4:

Now you are ready to save your work, compile (convert to 1s and 0s), and upload your code to the Arduino. To save navigate to the menu bar click “File”, then “Save As”. Name your program and save your program. Next step at the top by the File option select the “Check Mark”. This will compile your code and check for any errors before the program can be uploaded to the Arduino. Down at the bottom of the window the software will display possible errors (seek help on Arduino.cc Forum) or “Done Compiling”. If the software displayed “Done Compiling”, go back to the top and click on the Arrow next to the Check Mark. The software will recompile and write the program to the Arduino and the LED on the board should then start blinking twice per second.

Step 5:

You can modify this code as much as you want now. I would suggest starting with the “delay (500);” and changing the “500” to any value between 1 to 10000. Do not add non numerical characters to the number. After changing the value or values recompile and upload to the board. Watch how the LED will blink. If you would like to try other programs, select ‘File-Examples-Basic’ here you will find “blink” the program we just modified or select another program from the list. By searching Arduino.cc and the internet you can find more programs.

I hope you enjoyed this tutorial and have fun with your Arduino.
